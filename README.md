# Quote of the Day

A modern, minimal single-page website that displays one inspirational quote per day. Built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

## Features

- **Daily quote rotation** — a new quote is selected each day based on the day of the year
- **Clean, minimal design** — centered layout with elegant serif typography (Playfair Display) and generous whitespace
- **Dark / Light mode** — follows system preference with a manual toggle
- **Copy & Share** — one-click copy-to-clipboard and native Web Share API support
- **Keyboard shortcuts** — press `Space` or `→` to load a new random quote
- **Smooth animations** — subtle fade and slide transitions on quote change
- **Fully responsive** — works beautifully on mobile, tablet, and desktop

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [next-themes](https://github.com/pacocoursey/next-themes)
- [Lucide Icons](https://lucide.dev/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```
